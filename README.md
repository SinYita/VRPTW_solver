# VRPTW Solver - 彩色终端2D分层模型可视化

这是一个使用C++实现的彩色终端可视化工具，用于展示分层2D模型，其中每层代表一个特定的时间窗口。

## 功能特性

- 🎨 **彩色终端输出**: 使用ANSI颜色代码创建美观的彩色显示
- 📊 **分层2D可视化**: 每个时间窗口作为独立的2D表格层
- ⏱️ **时间序列动画**: 按时间窗口顺序播放动画
- 🎯 **随机点生成**: 自动生成带有坐标和时间戳的点数据
- 🔄 **交互模式**: 支持手动控制时间窗口切换
- 📋 **详细统计**: 显示每个时间窗口的点数和位置信息

## 编译和运行

### 方法1：使用Makefile（推荐）

```bash
# 编译程序
make

# 编译并运行
make run

# 清理编译文件
make clean

# 查看帮助
make help
```

### 方法2：直接使用g++

```bash
# 编译
g++ -std=c++17 -Wall -Wextra -O2 -o terminal_viz colorful_terminal_viz.cpp

# 运行
./terminal_viz
```

## 使用说明

### 运行模式

程序提供两种运行模式：

1. **自动播放模式**: 自动按时间窗口顺序播放动画
2. **交互模式**: 手动控制时间窗口切换

### 交互控制

在交互模式下，可以使用以下命令：
- `n` 或 `N`: 切换到下一个时间窗口
- `p` 或 `P`: 切换到上一个时间窗口  
- `a` 或 `A`: 开始自动播放动画
- `q` 或 `Q`: 退出程序

### 显示说明

- 每个点用不同的彩色符号表示
- 网格背景用点号(·)填充
- 每个时间窗口有独特的颜色和符号组合
- 显示当前时间窗口的统计信息

## 配置参数

程序中可以调整以下参数：

```cpp
const int GRID_WIDTH = 20;    // 网格宽度
const int GRID_HEIGHT = 12;   // 网格高度  
const int TIME_WINDOWS = 5;   // 时间窗口数量
```

## 技术特性

- **C++17标准**: 使用现代C++特性
- **ANSI颜色支持**: 兼容大多数现代终端
- **随机数生成**: 使用Mersenne Twister算法
- **内存安全**: 使用智能指针和现代C++实践
- **跨平台兼容**: 支持Linux、macOS和Windows

## 依赖要求

- C++17兼容的编译器（g++ 7.0+, clang++ 5.0+）
- 支持ANSI颜色代码的终端
- 标准C++库
